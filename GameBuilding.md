1. 나열한 기능들을 묶어 본다 (클래스)
2. class간 연결해 본다. 

- 숫자 야구를 만든다

- 기능
  - 겹치지 않는 4자리 숫자가 정해진다.  
  - player가 4자리 숫자를 입력한다.  
  - 자리수와 숫자가 맞으면 스트라이크 숫자만 맞으면 볼이 나온다.  
  (혹시 player가 입력을 겹치는 숫자를 입력할 경우 스트라이크와 볼 판단 똑바로) 
  - 맞추면 Game clear, 일정 횟수가 틀리면 game over가 뜬다.

- 부가기능
  - 총 9이닝으로 진행하며 이닝 당 3번의 기회가 주어진다
  - 현재 이닝과 남은 기회 수를 출력한다
  - 게임 스타트 화면과 게임 진행 화면을 출력한다
  - 플레이 방법을 시작화면에서 볼 수 있게한다
  - 4자리 숫자만 입력 받을 수 있게

- 클래스로 나누기
  - 숫자가 정해지는 클래스
  - 숫자를 입력받는 클래스
  - 스트라이크와 숫자가 정해지는 클래스
  - 게임 클리어를 판단하는 클래스
  - 게임 화면을 보여주는 클래스들 (시작화면, 플레이 화면)

- 클래스간 연결하기
  - 게임 클리어를 판단하는 클래스, 숫자가 정해지는 클래스, 숫자를 입력받는 클래스, 스트라이크와 숫자가 정해지는 클래스, 플레이 화면을 보여주는 클래스, 클리어 화면을 보여주는 클래스
  - 시작화면을 보여주는 클래스, 플레이 화면을 보여주는 클래스
