## How to play
1. 중복되지 않는 4자리 숫자를 입력한다.

2. 입력된 수의 자릿수와 숫자가 모두 맞은 수만 큼 스트라이크 처리한다.

3. 입력된 수의 숫자만 맞은 수만 큼 볼 처리한다.

4. 스트라이크와 볼이 0개일 때는 아웃 처리한다.

5. 총 9이닝으로 이닝 당 3번의 기회가 주어진다.

6. 스트라이크 4개가 나오면 승리한다.
